What is in the zip?

      
A shapefile is a set of related files: .shp stores geometry, .dbf stores attributes, .prj describes the coordinate system, and .cpg can specify text encoding. This page uses fetch() and shpjs to read them into WGS84 GeoJSON entirely in the browser.
