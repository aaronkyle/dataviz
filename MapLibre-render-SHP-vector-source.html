Standalone MapLibre template

Open zipped shapefiles.

This single HTML file downloads two SHP archives, converts them to GeoJSON, and layers county polygons with DCR trail lines—without a GIS server or access token.

  • ZIP → GeoJSON
  • Multiple layers
  • No access token

How it works

  1. Fetch each zipped shapefile as an ArrayBuffer.
  2. Parse and reproject the archives with shpjs.
  3. Style polygon and line collections as separate MapLibre layers.
To reuse the template, edit DATASETS and the preferred layer names near the top of the script. A typical ZIP includes matching .shp, .dbf, and .prj files.
