Why this works

Placement stays in the stylesheet.

The content remains in a logical document order. CSS changes only how much horizontal emphasis each region receives.

.feature {
  grid-column: 1 / span 7;
}

.placement {
  grid-column: 8 / -1;
}

Responsive behavior

The reading order remains predictable

Visual placement never replaces semantic order. If Grid is unsupported, styles fail, or the viewport narrows, the introduction still precedes the example, and the example still precedes its measures and notes.
